# KO掌机助手 下载站

这是 KO掌机助手 的 GitHub Pages 下载页面。

## 设置 GitHub Pages

1. 进入仓库 Settings → Pages
2. Source 选择 `Deploy from a branch`
3. Branch 选择 `main`，文件夹选择 `/docs`
4. 点击 Save

## 发布 Release

下载链接指向 GitHub Releases，需要先创建 Release：

1. 进入仓库 → Releases → Create a new release
2. Tag 填写 `v1.0.0`
3. 上传以下文件：
   - `KO掌机助手_1.0.0_x64-setup.exe` (安装版)
   - `KO掌机助手_1.0.0_绿色版.zip` (绿色版)
4. 发布 Release

## 访问地址

设置完成后，访问：`https://koboling.github.io/gpd-control-panel/`
